İdarəetmə Alqoritmi

Cihaz hər 100 millisaniyədən bir ətraf mühitdən gələn səs dalğalarını rəqəmsal verilənlərə çevirir.

1

Normal İş Rejimi

Səs < 20: Sakit dialoq və fərdi iş.

2

Müzakirə Rejimi

20-60: Qrup işləri və aktiv dərs.

3

Kritik Limit

Səs > 60: Buzzer və qırmızı LED aktivləşir.

// Sensor oxunuşu
int səs = analogRead(A0);

// Məntiq
if (səs > 600) {
  digitalWrite(LED_RED, HIGH);
  tone(BUZZER, 1000); 
} else if (səs > 200) {
  digitalWrite(LED_YELLOW, HIGH);
  noTone(BUZZER);
} else {
  digitalWrite(LED_GREEN, HIGH);
}

Tez-tez Verilən Suallar

Sensorun həssaslığını necə tənzimləmək olar?

Sensorun üzərindəki potensialometr vasitəsilə və ya kod bölməsindəki eşik (threshold) rəqəmlərini dəyişməklə istənilən mühitə uyğunlaşdırmaq mümkündür.

Layihənin gələcək inkişafı nədir?

Gələcəkdə Wi-Fi modul əlavə edərək gün ərzində səs səviyyəsini bulud sisteminə köçürmək və həftəlik statistika çıxarmaq planlaşdırılır.

Buzzer həmişə çalır?

Xeyr, buzzer yalnız səs səviyyəsi 70-dən yuxarı qalxdıqda və sinifdə kəskin küy olduqda xəbərdarlıq məqsədilə işə düşür.
