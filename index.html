Səs Svetoforu

Sinfin Ağıllı Keşikçisi

❓ Niyə Səssizlik Lazımdır?

Məktəb təkcə bilik aldığımız məkan deyil, həm də birgəyaşayış qaydalarını öyrəndiyimiz sosial mühitdir. Lakin idarəolunmaz səs-küy diqqəti dağıdır, öyrənmə prosesini ləngidir və müəllimin enerjisini tükədir.

Bu interaktiv tətbiq, səsi rəqəmlərə, rəqəmləri isə rənglərə çevirən "Səs Svetoforu" layihəsinin iş prinsipini nümayiş etdirir. Məqsəd, texnoloji həll yolu ilə intizamı bərpa etmək və şagirdlərdə özünüidarəetmə vərdişi yaratmaqdır.

“ Məqsədimiz səsi rəqəmlərə, rəqəmləri isə rənglərə çevirməkdir! ”

Simulyator: Özün Yoxla

Aşağıdakı çubuğu sürüşdürərək sinifdəki səs səviyyəsini dəyişin və cihazın necə reaksiya verdiyini izləyin.

İş Prinsipi və Məntiqi Model

Cihaz ətraf mühiti dinləyir və proqramlaşdırılmış məntiqə əsasən 3 əsas zonada reaksiya verir. Bu, Arduino Uno beyninə yazılmış sadə alqoritmlə tənzimlənir.

🟢

Yaşıl Zona (Sakitlik)

< 20 vahid

🟡

Sarı Zona (Xəbərdarlıq)

20 - 60 vahid

🔴

Qırmızı Zona (Həyəcan)

> 60 vahid

+ Buzzer səs siqnalı

// Alqoritmik Məntiq

ƏGƏR (Səs < 20) { 
  Yaşıl işığı yandır;
  Buzzer(Söndür);
}
ƏGƏR (Səs >= 20 VƏ Səs <= 60) { 
  Sarı işığı yandır;
  Buzzer(Söndür);
}
ƏGƏR (Səs > 60) { 
  Qırmızı işığı yandır; 
  Buzzer(Səs_Siqnalı_Ver);
}

Ağıllı Məktəbə Doğru

"Səs Svetoforu" sadəcə bir cihaz deyil, o, bizə özümüzü idarə etməyi öyrədir. Bu cihaz olan otaqda müəllimin 'sakit olun' deməsinə ehtiyac qalmır, çünki texnologiya bizə səhvimizi nəzakətlə, işıqlarla göstərir. Gəlin dərslərimizi daha texnoloji, siniflərimizi daha nizamlı edək!
